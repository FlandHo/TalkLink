# 欢迎使用Fland Studios的TalkLink软件
这个软件的代码使用GPL协议，各位可以修改和添加功能

你也可以把它当作一个私人的聊天软件，你可以自定服务器等设置

当然，如果你想学习Tkinker和socket，这些代码也是个例子

不过，请不要将此项目作恶意用途
# 使用方法
### 1.直接使用(连接官方服务器)
请下载Compiler文件夹中的exe文件，完成后执行
### 2.自建聊天环境
请下载所有文件后启动Server.py文件(你可以修改端口号等的信息)

如果是同机器测试，请将ChatMain.py中的"fland.zh.kg"改为127.0.0.1端口号要和服务端文件相同

如果为长期使用，请将ChatMain中的IP修改为自己服务器的IP，端口则为服务端配置的端口/公网访问端口
